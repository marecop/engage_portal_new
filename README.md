# ULink College Student Portal New
欢迎大家，我是Flaps1+F。此项目为ULink College的学生与老师门户Engage门户的学生端重制版，由于原来学生端的前端实在是太嗨儿丑了，而且屎山代码堆积而成的，于是我试着制作了这个前端页面，并且成功换壳。
## Features
### 原网页
首先，整个门户网站架构是通过ASP.NET WebForms组成的，而且系统架构非常老式，页面还在使用.aspx，接口还在使用.asmx，这种都快30年前的产物到现在还在用，实在是太罕见了，这样的架构到现在几乎绝迹。我在想，制作这个系统的公司难道没有能力更新一下架构的吗？
### 关于此项目
此项目通过Express+Vite打造成了全栈网页。页面更现代，更美观，速度更快。还不用担心那大头照太丑展示出来太尴尬。我属实不知道为什么学校拍这么丑的照片，这个系统还放大头照？？
### 换壳实现方式
非常简单，通过在ulinkcollege.engagehosted.cn进行任何对页面的操作，并从浏览器的dev tools里抓取到前端往后端发送的请求，过滤一下请求就可以找到所有的api链接。dev tools也提供了查看请求的详细信息的功能，查看请求头和请求体就可以轻松复刻原网页发送请求的格式和所需字段。
### 跨域
浏览器会限制跨域请求，于是写了一个代理程序用来转发这个项目的请求，很简单的解决跨域问题。
### 新网页内容
其实主要内容与原来的无异，主要内容是相同的，改动的还是样式外观的设计，而且大幅优化了代码，不再用一大堆css来套了。
### 成绩页面
在Engage后端架构里，查询成绩的api是一个嵌套的api，我一开始还没发现，直到仔细看请求体才发现是一堆嵌套的……不过解决方式很简单。
## 结语
很简单的项目，大家可以自己下下来，从终端进入根目录，运行npm install之后再运行npm run start可以启动生产服务器。当然，也可以访问我们的页面<a href="https://portal.flaps1f.com">portal.flaps1f.com</a>，使用自己engage的账号密码登录来尝试一下体验如何，**注意：一定要先确认好自己的账号和密码是否正确，是否曾输入错过密码，显示无效账号或密码时，一定要确认是否输错，如果账号和密码没有问题，可能账户因为密码输错太多，要回去engage原网页解锁。**。此项目目前会持续更新。
